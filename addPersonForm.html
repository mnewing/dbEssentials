Usage notes

This page is the form to collect the data. when the submit button is pressed it will load the addPerson page that will do the insert.

Validation

mandatory fields are marked with a * and set to required - browsers will ensure that these fields are not left empty

fields types are appropiate to the data being collected. age is a number and some sensible mins and maxs have been set. modern browsers will ensure that only numbers between 18 and 120 are entered

Add

To add to the database fill in the form and press submit.

Required fields are marked with the * symbol.
